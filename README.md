# Football Team Manager ⚽

A Flutter application for managing football (soccer) teams with Firebase backend integration. This app provides team management, player organization, and user authentication features with a modern, responsive UI.

## 📱 Overview

Football Team Manager is a cross-platform mobile application built with Flutter that enables users to manage football teams efficiently. The application features user authentication, real-time data synchronization with Firebase, and a beautiful Material Design interface with dark mode support.

## ✨ Features

- **User Authentication**: Firebase authentication with email/password login and signup
- **Real-time Database**: Firebase Realtime Database for instant data synchronization
- **Team Management**: Create and manage multiple football teams
- **Player Organization**: Add, edit, and organize team players
- **Dark Mode Support**: Full dark and light theme support with user preference persistence
- **Cross-Platform**: Available on Android, iOS, Web, Windows, macOS, and Linux
- **Responsive Design**: Optimized UI for various screen sizes and devices
- **Custom Styling**: Professional color scheme with teal and green accent colors
- **Icon Support**: Comprehensive icon library using Font Awesome
